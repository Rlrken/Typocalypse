# Typocalypse
Typocalypse - Type Fast, Die Last

Instructions!!!!

1. Download the Typocalypse folder and open it
2. Download the fonts to implement the fonts used
3. Open the src folder
4. Use only JGRASP to run the program because other IDE does not follow the same directory format as JGRASP
5. Find the GameWindow.java and run it
6. Enjoy the game and goodluck! :>>>
